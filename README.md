# Financial KPI Monitoring & Investment Insights Dashboard

## Project Overview

This project focuses on monitoring financial performance and investment efficiency using Tableau. The dashboard provides a comprehensive view of revenue, profit, ROI, portfolio value, and investment performance across different regions and categories.

---

## Objective

The objective of this project is to:

* Monitor key financial KPIs
* Analyze investment performance
* Track profitability trends
* Compare budgeted vs actual performance
* Support financial and strategic decision-making

---

## Tools Used

* Tableau Public
* Microsoft Excel
* Financial Analytics
* Business Intelligence
* Data Visualization

---
